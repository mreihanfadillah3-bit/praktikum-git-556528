Git dan GitHub

1. Apa Itu Git?

Git adalah sebuah Version Control System (VCS) lokal yang digunakan untuk melacak perubahan pada file proyek. Dengan Git, Anda bisa kembali ke versi kode sebelumnya jika terjadi kesalahan.

2. Apa Itu GitHub?

GitHub adalah platform berbasis cloud yang digunakan untuk menyimpan dan mengelola repositori Git secara online. GitHub memudahkan kolaborasi antar pengembang di seluruh dunia.

3. Perintah Dasar Git

Berikut merupakan tabel perintah yang paling sering digunakan:

Perintah Kegunaan
git init Membuat repositori baru di folder lokal.
git add . Menyiapkan semua perubahan file untuk disimpan (staging).
git commit -m "pesan" Menyimpan perubahan ke dalam riwayat Git.
git push Mengirim perubahan dari lokal ke GitHub.
git pull Mengambil dan menggabungkan perubahan dari GitHub ke lokal.

4. Alur Kerja Sederhana

  1. Buat akun di GitHub.com.
  2. Install Git di komputer Anda.
  3. Gunakan git config untuk mengatur nama dan email.
  4. Buat repositori baru di GitHub, lalu hubungkan dengan perintah git remote add origin.
